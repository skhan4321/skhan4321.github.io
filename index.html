Summary: Latitude vs. x

The purpose of this project was to analyze weather data in multiple cities and determine if there is a relationship between climate and proximity to the equator. To do so, we used the OpenWeather data API to extract 500 cities. From there we used Matplotlib to plot the data analyze based on four different criteria.
